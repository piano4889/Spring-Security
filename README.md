# 개요
본 프로젝트는 Spring Security 및 JWT 구현해서 api 호출 시 jwt 검증까지 확인하는 프로젝트입니다.

## 개발 환경
* Java 1.8
* IntelliJ
* SpringBoot 2.7.7
* Maria DB , Redis
* MyBatis

## 주요 스택
*  Spring Security
*  Spring Mail
*  log4j2
*  JsonWebtoken(JWT)
*  Lombok
*  swagger
*  pagehelper

# 주요 기능
* 회원 가입
  * 일반 로그인 - Oauth2(Kakao)
* JWT 기반 API 호출
  * 회원 - 비회원 구분
* 게시글, 댓글 CRUD
* File Upload - Download
